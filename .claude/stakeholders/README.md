# Stakeholders Directory

This directory contains stakeholder information and interaction logs managed by the Stakeholder Manager agent.

## File Structure

Each stakeholder has their own markdown file named: `familyname_firstname.md`

## Usage

Use the `/stakeholder-manager` command to interact with the Stakeholder Manager agent for stakeholder management operations.

See `.claude/skills/stakeholder-management.md` for the complete stakeholder management skill documentation.
